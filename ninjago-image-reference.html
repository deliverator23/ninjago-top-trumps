How to use this sheet:
  1. Click the Wiki Page button for a character — opens their Ninjago Wiki page in a new tab
  2. On the wiki page, right-click the character portrait (top-right infobox image) → Save image as…
  3. Save it with the exact filename shown (e.g. lloyd.jpg)
  4. Put all saved images in a folder called images/ next to the print HTML file
  5. Target crop size: 350 × 180 px at 150 dpi — landscape, top of image anchored
